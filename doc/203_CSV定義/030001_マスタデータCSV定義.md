## 3. CSV詳細

### 3.1. データ取得対象路線CSV
実際に Places API からデータを取得する路線の
路線コードを記載するCSVデータ。

**ファイル名**
mst_fetch_target_line.csv

**CSV仕様**
種別コード,対象コード,補足情報（メモ）

**データ仕様**
- 種別コード：
  - 1 … 会社コードでの指定
  - 2 … 路線コードでの指定
- 対象コード
  - 種別コードが 1 の場合は、取得対象となる会社の company_cd を指定する。
  - 種別コードが 2 の場合は、取得対象となる路線の line_cd を指定する。

**その他**
- ヘッダーあり
- 補足情報（メモ）は処理のデータとしては使用されない


### 3.2. 取得対象チェーンCSV
Place API で実際に検索に使用するカフェチェーンの

- ヘッダーあり
